<p>At Build 2025, SQL Server 2025 officially entered&nbsp;<a href="https://www.microsoft.com/en-us/sql-server/blog/2025/05/19/announcing-sql-server-2025-preview-the-ai-ready-enterprise-database-from-ground-to-cloud/" rel="noreferrer noopener" target="_blank">public preview</a>. As one of the world’s most popular databases, this release continues a decades-long history of innovation with features made for developers, AI, analytics, and cloud connectivity. SQL Server 2025 is the most significant release for SQL developers in the last decade and will help streamline application development and greatly reduce complexity. With built-in vector support, you can now boost search intelligence by combining semantic search alongside full text search and filtering. This will allow you to run the generative AI models of your choice with your own data. SQL Server 2025 enables zero-ETL (extract, transform, and load), real-time analytics through mirroring in Microsoft Fabric and provides cloud agility through Azure Arc.</p>

<h2 class="wp-block-heading" id="a-thriving-community-at-the-heart-of-sql-server-2025">A thriving community at the heart of SQL Server 2025</h2>



<p>As we look back at the creation of SQL Server 2025, we want to extend our deepest gratitude to the community leaders, MVPs, partners, and Microsoft engineers who have shared their insights, expertise, and enthusiasm about this exciting release. Their passion for SQL Server and commitment to knowledge-sharing continue to inspire us.</p>



<p>In the resources below we showcase the expertise of our community as well as Microsoft, with blogs and videos that will help you make the most of everything SQL Server 2025 has to offer. These resources dive into key innovations that set SQL Server 2025 apart, including best-in-class security, built-in AI, developer enhancements, and Fabric mirroring.</p>

<h3 class="wp-block-heading" id="made-for-developers">Made for developers</h3>



<p>SQL Server 2025 is the most significant release for SQL developers in the last decade. With features including native JSON support, built-in REST APIs and RegEx enablement, change event streaming plus a new Standard Developer edition and new SSMS release, SQL Server 2025 helps streamline application development and greatly reduce code complexity.</p>



<ul class="wp-block-list">
<li>SQL Server Management Studio (SSMS) 21 is now generally available. Read about the new SSMS release in the <a href="https://techcommunity.microsoft.com/blog/sqlserver/sql-server-management-studio-ssms-21-is-now-generally-available-ga/4415230" rel="noreferrer noopener" target="_blank">Microsoft Community Hub</a>.</li>



<li>Copilot is available in SSMS, learn how <a href="https://techcommunity.microsoft.com/blog/sqlserver/announcing-the-preview-of-copilot-in-sql-server-management-studio-ssms/4415267" rel="noreferrer noopener" target="_blank">Copilot in SSMS preview</a> can help make you more efficient – answering your questions and helping you write, edit, and fix T-SQL.</li>



<li>JSON Enhancements are covered by MVP Daniel Hutmacher who shares hands-on insights in <a href="https://sqlsunday.com/2025/05/19/json-indexes-first-impressions/" rel="noreferrer noopener" target="_blank">JSON indexes in SQL Server: First impressions</a>, while Microsoft SQL Engineering PM Umachandar Jayachandran explores <a href="https://devblogs.microsoft.com/azure-sql/announcing-the-general-availability-ga-of-json-data-type-json-aggregates/">JSON data types and aggregates</a> in his blog.</li>



<li>SQL Server 2025 introduces native support for regex functions, making it easier to perform complex text operations directly within SQL queries.&nbsp;Read Microsoft SQL Engineering PM Abhiman Tiwari’s blog to <a href="https://devblogs.microsoft.com/azure-sql/unlocking-the-power-of-regex-in-sql-server/" rel="noreferrer noopener" target="_blank">Unlock the Power of Regex in SQL Server</a>.</li>
</ul>

<li>And for our Spanish speakers, Javier Villegas, Microsoft MVP, has created a wealth of in-depth content including his video on <a href="https://www.youtube.com/watch?v=wENLJDUqvTM" rel="noreferrer noopener" target="_blank">Mirroring in Microsoft Fabric &#8211; SQL Server</a>.</li>
