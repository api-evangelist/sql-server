# Microsoft SQL Server APIs

> A collection of APIs, drivers, and programmable interfaces for interacting with Microsoft SQL Server and Azure SQL — spanning the native database engine (T-SQL), Azure management REST APIs, connectivity drivers for every major language, and newer agent-native surfaces (Data API builder REST/GraphQL and the SQL MCP Server). Generated by API Evangelist from the provider's public catalog.

## APIs

- [SQL Server Database Engine API](https://docs.microsoft.com/en-us/sql/sql-server/): Native T-SQL database engine for connecting and executing queries.
- [SQL Server REST API](https://docs.microsoft.com/en-us/rest/api/sql/): Azure Resource Manager REST API for managing SQL resources (base management.azure.com).
- [Azure SQL Database REST API](https://docs.microsoft.com/en-us/azure/azure-sql/database/): Manage Azure SQL Database resources.
- [Azure SQL Managed Instance REST API](https://learn.microsoft.com/en-us/azure/azure-sql/managed-instance/): Create, configure and manage Managed Instances.
- [SQL Server Reporting Services (SSRS) REST API](https://docs.microsoft.com/en-us/sql/reporting-services/developer/rest-api): Manage and access reporting services.
- [SQL Server Management Objects (SMO)](https://docs.microsoft.com/en-us/sql/relational-databases/server-management-objects-smo/): .NET API for managing instances and databases.
- [SQL Server Analysis Services (SSAS) API](https://docs.microsoft.com/en-us/analysis-services/): Manage and query Analysis Services.
- [SQL Server Integration Services (SSIS) API](https://learn.microsoft.com/en-us/sql/integration-services/sql-server-integration-services): Programmable ETL object model.
- [External REST Endpoint Invocation](https://learn.microsoft.com/en-us/sql/relational-databases/system-stored-procedures/sp-invoke-external-rest-endpoint-transact-sql): sp_invoke_external_rest_endpoint calls external HTTPS REST endpoints from T-SQL.
- [Data API Builder](https://learn.microsoft.com/en-us/azure/data-api-builder/): Auto-generate REST and GraphQL endpoints (and an MCP server) over SQL Server.

## Drivers

- [ODBC Driver for SQL Server](https://docs.microsoft.com/en-us/sql/connect/odbc/): Cross-platform ODBC connectivity.
- [JDBC Driver (mssql-jdbc)](https://learn.microsoft.com/en-us/sql/connect/jdbc/): Java connectivity.
- [ADO.NET Provider (Microsoft.Data.SqlClient)](https://learn.microsoft.com/en-us/sql/connect/ado-net/microsoft-ado-net-sql-server): Official .NET provider.
- [OLE DB Driver](https://learn.microsoft.com/en-us/sql/connect/oledb/oledb-driver-for-sql-server): Windows OLE DB access.
- [Node.js Driver (mssql / tedious)](https://learn.microsoft.com/en-us/sql/connect/node-js/node-js-driver-for-sql-server): JavaScript/TypeScript connectivity.
- [Python Drivers (mssql-python, pyodbc, pymssql)](https://learn.microsoft.com/en-us/sql/connect/python/python-driver-for-sql-server): Python connectivity.
- [Go Driver (go-mssqldb)](https://learn.microsoft.com/en-us/sql/connect/golang/microsoft-go-mssqldb-driver): Go connectivity.
- [PHP Drivers (SQLSRV / PDO_SQLSRV)](https://learn.microsoft.com/en-us/sql/connect/php/microsoft-php-driver-for-sql-server): PHP connectivity.
- [Ruby Driver (TinyTDS)](https://learn.microsoft.com/en-us/sql/connect/ruby/step-3-proof-of-concept-connecting-to-sql-using-ruby): Ruby connectivity.
- [Entity Framework Core SQL Server Provider](https://learn.microsoft.com/en-us/ef/core/providers/sql-server/): .NET ORM.

## Agent-native

- [SQL MCP Server](https://learn.microsoft.com/en-us/azure/data-api-builder/mcp/overview): Official open-source MCP server (in Data API builder) exposing seven DML tools over SQL Server with RBAC.
